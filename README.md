# Lingua Franca

A Relationship-based Access Control service for 2nd semester university project at AAU.

# Commit style
We use [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/), i.e:
- `feat(scope): message` for new features and additions
- `fix(scope): message` for bugfixes and the likes

### Examples:
- `feat(ACL): Added graph search`
- `fix(serialization): ACL's now load correctly on startup`
